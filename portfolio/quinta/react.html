React Js

15.10.18

per tre giorni abbiamo partecipato ad un corso di React JS, una libreria di javascript creata da Facebook per sviluppare interfacce utente, per siti web o applicazioni mobile. marco argentieri

Primo giorno

Subito ci sono state spiegate le basi di react, come installarlo e la sintassi di base.
Oltre a react, ci è stata mostrata l'importanza delle API (Application Programming Interface) e l'utilità di Github, una piattaforma per sviluppatori dove è possibile lavorare in gruppo a progetti di informatica in modo semplice e ottimale.

Come primo progettino ci è stato proposto di sviluppare un'app in react che utilizzasse le API del meteo per mostrare le informazioni basi del tempo data una città in input.
qui il link all'esercizio sul mio profilo Github.

Secondo giorno

Ci è stato proposto di mettere in pratica le conoscenze acquisite tramite un progetto di gruppo:
l'obiettivo era quello di sviluppare un'app in react che utilizzasse un'API qualsiasi a scelta trovata su internet(ovviamente gratis ed utilizzabile).

Il mio gruppo era composto dai miei compagni Caraffi Alex, Serafino Niccolò, Perna Mario e Betuzzi Gabriele.
Abbiamo sviluppato un'applicazione che permette di visualizzare il testo di una canzone, dato il suo titolo e il nome dell'autore.
Purtroppo, le API che lavorano in questo campo sono tutte a pagamento, tranne per MusixMatch che permette di visualizzare il 40% del testo con la versione gratuita.

Per realizzarlo in gruppo abbiamo imparato a usare correttamente le funzioni di gruppo di Github, che ci è stato davvero molto utile per la gestione dei cambiamenti e degli aggiornamenti del codice.

Terzo Giorno

L'ultimo giorno dell'esperienza è stato usato per concludere il progetto, e per chiedere le ultime domande sul codice al referente dell'attività.
Infine, la giornata si è conclusa con l'esposizione da parte di alcuni gruppi dei vari progetti.
qui il link al progetto finale sul mio profilo Github.
